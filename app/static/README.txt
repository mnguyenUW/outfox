Healthcare Cost Navigator UI

How to use:
1. Start your FastAPI app (e.g., `uvicorn app.main:app --reload`).
2. Open your browser and go to: http://localhost:8000/static/index.html
   - The Providers section will display data from the `/providers` endpoint.
   - The Ask section lets you send questions to the `/ask` endpoint and view responses.

Notes:
- The UI is a minimal demo using plain HTML and JavaScript.
- If you update the backend endpoints, reload the page to see changes.
